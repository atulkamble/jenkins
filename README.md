# Jenkins Complete Course

A comprehensive Jenkins course with hands-on examples, code samples, and practical projects.

## Course Overview

This course is designed to take you from Jenkins beginner to advanced practitioner. You'll learn through hands-on exercises, real-world examples, and practical projects.

## Prerequisites

- Basic understanding of software development
- Familiarity with command line/terminal
- Basic knowledge of Git
- Understanding of basic DevOps concepts

## Course Structure

### Module 1: Jenkins Fundamentals
- Introduction to Jenkins and CI/CD
- Installation and setup
- Jenkins architecture and components
- Basic configuration

### Module 2: Jenkins Jobs and Builds
- Creating your first Jenkins job
- Build triggers and scheduling
- Build parameters and environment variables
- Build history and artifacts

### Module 3: Jenkins Pipelines
- Introduction to Jenkins Pipelines
- Declarative vs Scripted pipelines
- Pipeline syntax and structure
- Pipeline best practices

### Module 4: Source Code Management
- Git integration
- Repository webhooks
- Branch strategies
- Multi-branch pipelines

### Module 5: Build Tools Integration
- Maven integration
- Gradle integration
- Node.js projects
- Docker builds

### Module 6: Testing and Quality Gates
- Unit test integration
- Code quality analysis with SonarQube
- Test reporting
- Quality gates and build failures

### Module 7: Deployment and Delivery
- Deployment strategies
- Environment management
- Blue-green deployments
- Rolling deployments

### Module 8: Jenkins Security
- User management and authentication
- Authorization strategies
- Security best practices
- Credential management

### Module 9: Jenkins Administration
- Plugin management
- System configuration
- Backup and restore
- Performance monitoring

### Module 10: Advanced Topics
- Jenkins as Code (JCasC)
- Distributed builds and agents
- Pipeline libraries
- Jenkins REST API

## Getting Started

1. Navigate to [Module 1](./module-01-fundamentals/) to begin
2. Follow the modules in order for best learning experience
3. Complete all hands-on exercises
4. Work on the capstone project in Module 10

## Course Duration

- **Estimated Time**: 40-50 hours
- **Self-paced learning**
- **Hands-on exercises**: 60% of the course
- **Theory**: 40% of the course

## Support and Resources

- Check the `resources/` folder for additional materials
- Each module contains practical exercises
- Sample code and configurations are provided
